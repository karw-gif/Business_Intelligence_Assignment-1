# Business Intelligence Assignment 1 – Decision Tree Analysis

This project uses a **Decision Tree machine learning model** to perform classification and prediction tasks as part of a Business Intelligence course assignment.

## 📌 Objective

The goal is to build and evaluate a decision tree model to classify or predict outcomes based on structured input features from a real dataset. This analysis demonstrates key BI concepts including data preprocessing, model training, and performance evaluation.

## 🧠 Technologies & Methods

- Python (Jupyter Notebook)
- Pandas for data manipulation
- Scikit‑learn for machine learning
- Decision Tree Classifier
- Data visualization (charts, graphs)

## ✨ Key Steps

1. **Data Loading & Exploration**
   - Load and inspect dataset
   - Visualize distributions and relationships

2. **Data Cleaning & Preprocessing**
   - Handle missing values
   - Encode categorical variables as needed

3. **Model Building**
   - Train a decision tree classifier
   - Visualize tree structure

4. **Evaluation**
   - Model accuracy
   - Feature importance
   - Predictions on test samples

## 📈 Results & Insights

- The decision tree model showed reasonable classification performance.
- Top features impacting decisions were identified through feature importance.
